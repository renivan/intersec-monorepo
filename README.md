## 🏗️ Arquitetura de Missão

O ecossistema está dividido de forma modular para máxima eficiência:

1. **Native Core (Rust)**: Motor de captura bruta e análise de protocolos (DPI) de ultra-baixa latência.
2. **Android Guardian (Compose)**: Interface tática desenvolvida em Jetpack Compose com arquitetura MVVM.
3. **Sentinel Tunnel (VPN)**: Capacidade de inspeção de tráfego local sem necessidade de acesso ROOT.
4. **Cloud Intel (Firebase)**: Cérebro remoto para autenticação e atualizações de segurança.
